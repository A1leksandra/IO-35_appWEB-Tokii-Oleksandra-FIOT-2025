Створення HTML-коду головної сторінки WEB-застосунку

1. Опис предметного середовища

Предметне середовище нашого WEB-застосунку - це сфера електронної комерції (e-commerce) та продажу одягу та аксесуарів. Застосунок належить до галузі інтернет-торгівлі (Online Retail) та систем управління інтернет-магазинами (E-commerce Management Systems).

Основні характеристики предметного середовища:

  • Електронна комерція: Продаж одягу, аксесуарів та модних товарів через інтернет-платформу
  • Управління каталогом товарів: Ведення бази даних товарів з детальною інформацією про кожен продукт
  • Управління замовленнями: Обробка замовлень від моменту оформлення до доставки клієнту
  • Управління клієнтами: Ведення бази клієнтів, історія покупок, персоналізація рекомендацій
  • Система оплати та доставки: Інтеграція з платіжними системами та службами доставки

3. Структура головної сторінки WEB-застосунку

Головна сторінка WEB-застосунку (index.html) містить наступні основні елементи:

  • Заголовки різних рівнів (h1, h2, h3, h4) - для структурування контенту
  • Зображення - фотографії товарів, промо-матеріали, аватари
  • Таблиці - тарифи доставки, порівняльні таблиці
  • Списки - марковані та нумеровані списки переваг та інструкцій
  • Форма - форма зворотного зв'язку для контактів
  • Посилання - навігаційні посилання та посилання на сторінку з описом

4.3. Теорія про теги та атрибути таблиць

Основні теги для створення таблиць:

  • <table> - основний контейнер для таблиці. Атрибути:
    • border - товщина рамки (застарілий, використовується CSS)
    • cellpadding - відступ всередині комірок
    • cellspacing - відстань між комірками
    • width - ширина таблиці
  • <thead> - секція заголовків таблиці
  • <tbody> - тіло таблиці з даними
  • <tfoot> - підсумковий рядок таблиці
  • <tr> - рядок таблиці. Атрибути:
    • align - вирівнювання по горизонталі
    • valign - вирівнювання по вертикалі
  • <th> - комірка заголовка. Атрибути:
    • colspan - об'єднання комірок по горизонталі
    • rowspan - об'єднання комірок по вертикалі
    • scope - область дії заголовка (col, row, colgroup, rowgroup)
  • <td> - комірка з даними. Атрибути:
    • colspan - об'єднання комірок по горизонталі
    • rowspan - об'єднання комірок по вертикалі

5.3. Теорія про атрибути тегу <img>

Основні атрибути тегу <img>:

  • src (обов'язковий) - шлях до файлу зображення або URL. Може бути:
    • Відносний шлях: src="images/photo.jpg"
    • Абсолютний шлях: src="/images/photo.jpg"
    • URL: src="https://example.com/image.jpg"
  • alt (рекомендований) - альтернативний текст, який відображається, якщо зображення не завантажилося. Важливий для доступності та SEO
  • width - ширина зображення в пікселях або відсотках
  • height - висота зображення в пікселях або відсотках
  • title - текст підказки, що з'являється при наведенні миші
  • loading - режим завантаження:
    • lazy - відкладене завантаження (для оптимізації)
    • eager - негайне завантаження (за замовчуванням)
  • onerror - обробник помилки завантаження (JavaScript)
  • class - CSS клас для стилізації
  • id - унікальний ідентифікатор

6.4. Теорія про теги та атрибути списків

Марковані списки (<ul>):

  • <ul> - контейнер для маркованого списку. Атрибути:
    • type - тип маркера (disc, circle, square) - застарілий, використовується CSS
  • <li> - елемент списку. Може містити:
    • Текст
    • Інші HTML-елементи
    • Вкладені списки

Нумеровані списки (<ol>):

  • <ol> - контейнер для нумерованого списку. Атрибути:
    • type - тип нумерації (1, A, a, I, i)
    • start - початкове значення нумерації
    • reversed - зворотна нумерація
  • <li> - елемент списку. Атрибути:
    • value - значення для конкретного елемента (тільки для <ol>)

Списки визначень (<dl>):

  • <dl> - список визначень
  • <dt> - термін
  • <dd> - визначення терміну

7. Висновки

Під час виконання лабораторної роботи №1 було створено HTML-код головної сторінки WEB-застосунку для інтернет-магазину одягу. В процесі роботи було опановано навички використання основних HTML-тегів та атрибутів.

Основні досягнення:

  1. Створено структуровану HTML-сторінку з використанням семантичних тегів
  2. Реалізовано таблицю тарифів доставки з використанням тегів <table>, <thead>, <tbody>, <tr>, <th>, <td>
  3. Додано зображення товарів з правильним використанням атрибутів тегу <img>
  4. Створено марковані та нумеровані списки для відображення переваг та інструкцій
  5. Реалізовано форму зворотного зв'язку з різними типами полів вводу
  6. Додано навігаційні посилання та посилання на сторінку з описом предметної галузі
  7. Застосовано CSS для стилізації та створення сучасного дизайну

Отримані знання та навички дозволяють створювати структуровані та функціональні WEB-сторінки з використанням стандартних HTML-елементів. Розуміння призначення та використання атрибутів тегів є важливим для створення доступних та оптимізованих веб-додатків.
