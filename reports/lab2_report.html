6. Висновки

Під час виконання лабораторної роботи №2 було опановано використання каскадних таблиць стилів (CSS) та різних типів селекторів для стилізації WEB-застосунку. В процесі роботи було застосовано селектори тегів, класів, ідентифікаторів, а також інші типи селекторів для створення сучасного та функціонального дизайну.

Основні досягнення:

  1. Опановано використання селекторів тегів для стилізації HTML-елементів
  2. Застосовано селектори класів для створення перевикористовуваних стилів
  3. Використано селектори ідентифікаторів для унікальних елементів
  4. Застосовано сусідні, дочірні селектори, селектор атрибута та універсальний селектор
  5. Реалізовано стилізацію текстових елементів: шрифти, контури, фон, колір тексту
  6. Застосовано CSS-властивості для таблиць та багаторівневих списків
  7. Використано просунуті CSS технології: Flexbox, Grid, Transform, Transition, псевдокласи
  8. Всі стилі винесено в окремий файл styles.css для кращої організації коду

Отримані знання та навички дозволяють створювати сучасні та стильові WEB-сторінки з використанням каскадних таблиць стилів. Розуміння різних типів селекторів та їх застосування є важливим для створення гнучких та підтримуваних стилів веб-додатків.
