2. ЗАВДАННЯ №1

З об'єкту medicines потрібно:

  • Отримати масив, в якому будуть лише назви препаратів
  • Прибрати медикаменти, в яких строк зберігання вже пройшов
  • У новому масиві відсортувати медикаменти у хронологічному порядку
  • Результат вивести у консоль
  • Застосувати стрілочні функції

2.1. Програмний код

// Об'єкт з медикаментами та їх датами зберігання
const medicines = {
    Агалгін: new Date("2022-05-01"),
    Ношпа: new Date("2025-07-02"),
    Альфахолін: new Date("2024-12-21"),
    Аспірин: new Date("2022-08-15"),
    Аспаркам: new Date("2024-04-18"),
    Парацетамол: new Date(), // Сьогоднішня дата
    Ібупрофен: new Date("2025-12-31"), // Майбутня дата
    Нурофен: new Date("2026-03-15"), // Майбутня дата
};

// Отримуємо поточну дату
const currentDate = new Date();

// Отримуємо масив з назвами препаратів та їх датами (стрілочна функція)
const getMedicineNames = (meds) => Object.keys(meds);

// Фільтруємо непросрочені препарати (стрілочна функція)
const filterValidMedicines = (meds) => {
    return Object.entries(meds)
        .filter(([name, expiryDate]) => expiryDate >= currentDate)
        .map(([name, expiryDate]) => ({ name, expiryDate }));
};

// Сортуємо медикаменти у хронологічному порядку (стрілочна функція)
const sortByDate = (medicinesArray) => {
    return medicinesArray.sort((a, b) => a.expiryDate - b.expiryDate);
};

// Отримуємо тільки назви з відсортованого масиву (стрілочна функція)
const getNamesOnly = (medicinesArray) => {
    return medicinesArray.map(med => med.name);
};

2.2. Пояснення коду

Основні елементи коду:

  • const getMedicineNames = (meds) => Object.keys(meds) - стрілочна функція для отримання масиву ключів об'єкта (назв препаратів). Синтаксис стрілочної функції: `(параметри) => вираз`
  • Object.keys() - метод для отримання масиву ключів об'єкта
  • Object.entries() - метод для отримання масиву пар [ключ, значення] з об'єкта
  • .filter(([name, expiryDate]) => expiryDate >= currentDate) - метод filter() зі стрілочною функцією для фільтрації непросрочених препаратів. Використовує деструктуризацію масиву
  • .map(([name, expiryDate]) => ({ name, expiryDate })) - метод map() для перетворення масиву пар у масив об'єктів
  • .sort((a, b) => a.expiryDate - b.expiryDate) - метод sort() зі стрілочною функцією для сортування за датою у хронологічному порядку
  • .map(med => med.name) - метод map() для отримання тільки назв препаратів
  • Стрілочні функції - короткий синтаксис для написання функцій: `(параметри) => вираз` або `(параметри) => { return вираз; }`

3. ЗАВДАННЯ №2

Перевірити, чи є кожен елемент масиву парним:

  • Дано масив чисел: [2, 4, 6, 8, 10]
  • Перевірити, чи всі елементи парні
  • Очікуваний результат: true
  • Використати стрілочні функції

3.1. Програмний код

// Дано масив чисел
const numbers = [2, 4, 6, 8, 10];

// Перевірка, чи є кожен елемент масиву парним (стрілочна функція)
const areAllEven = (arr) => {
    return arr.every(num => num % 2 === 0);
};

// Альтернативний варіант з використанням стрілочної функції безпосередньо
const areAllEvenShort = (arr) => arr.every(num => num % 2 === 0);

// Виконуємо перевірку
const result = areAllEven(numbers);
console.log("Масив чисел:", numbers);
console.log("Чи всі елементи парні?", result);
console.log("Очікуваний результат: true");

3.2. Пояснення коду

Основні елементи коду:

  • const areAllEven = (arr) => { ... } - стрілочна функція для перевірки, чи всі елементи масиву парні
  • arr.every() - метод масиву, який перевіряє, чи всі елементи відповідають заданій умові. Повертає true, якщо всі елементи проходять перевірку, інакше - false
  • num => num % 2 === 0 - стрілочна функція-предикат для перевірки парності числа. Використовує оператор модуло (%)
  • num % 2 === 0 - перевірка парності: якщо залишок від ділення на 2 дорівнює 0, то число парне
  • const areAllEvenShort = (arr) => arr.every(num => num % 2 === 0) - короткий варіант стрілочної функції без фігурних дужок (implicit return)
  • Метод every() - перевіряє всі елементи масиву і повертає true тільки якщо всі елементи проходять перевірку

4. Висновки

Під час виконання лабораторної роботи №5 було створено JavaScript-код, який демонструє роботу зі стрілочними функціями та методами масивів для обробки об'єктів, фільтрації даних, сортування та перевірки умов.

Основні досягнення:

  1. Опановано синтаксис стрілочних функцій (arrow functions) та їх переваги
  2. Використано метод Object.keys() для отримання ключів об'єкта
  3. Використано метод Object.entries() для отримання пар ключ-значення з об'єкта
  4. Опановано метод filter() для фільтрації елементів масиву за умовою
  5. Опановано метод map() для перетворення елементів масиву
  6. Опановано метод sort() для сортування масивів
  7. Опановано метод every() для перевірки, чи всі елементи відповідають умові
  8. Використано деструктуризацію масивів у стрілочних функціях
  9. Реалізовано обробку дат та перевірку термінів зберігання
  10. Опановано ланцюжок методів (method chaining) для послідовної обробки даних
  11. Зрозуміто переваги стрілочних функцій: коротший синтаксис, автоматичне прив'язування this

Отримані знання та навички дозволяють ефективно працювати з масивами та об'єктами в JavaScript, використовуючи сучасний синтаксис стрілочних функцій та вбудовані методи масивів. Розуміння принципів роботи з методами filter(), map(), sort(), every(), Object.keys(), Object.entries() та стрілочними функціями є важливим для створення елегантного та ефективного коду в JavaScript.
